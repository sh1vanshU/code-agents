# Observability

Tracing, profiling, debugging, and monitoring tools.

| Module | Description |
|--------|-------------|
| `otel.py` | OpenTelemetry integration |
| `telemetry.py` | Telemetry metrics collection |
| `profiler.py` | Code profiling tools |
| `debug_engine.py` | Interactive debugging engine |
| `stack_decoder.py` | Stack trace decoder and analysis |
| `log_analyzer.py` | Log pattern analysis |
| `log_investigator.py` | Log investigation and search |
| `live_tail.py` | Real-time log tailing |
| `recon_debug.py` | Reconnaissance debugging tools |
| `tracing_migration.py` | Distributed tracing migration helper |
| `call_chain.py` | Call chain analysis and visualization |
| `deadlock_detector.py` | Deadlock detection and analysis |
| `leak_finder.py` | Memory/resource leak detection |
| `health_dashboard.py` | System health dashboard |
| `performance.py` | Performance analysis tools |
