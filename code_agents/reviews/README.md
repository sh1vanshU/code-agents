# Reviews

Code review, quality analysis, and style enforcement tools.

## Modules

| Module | Description |
|--------|-------------|
| `code_review.py` | Comprehensive code review engine |
| `review_buddy.py` | Friendly code review assistant |
| `review_autofix.py` | Auto-fix suggestions for review findings |
| `review_autopilot.py` | Automated review pipeline |
| `review_checklist.py` | Review checklist generation |
| `review_responder.py` | Auto-respond to review comments |
| `code_smell.py` | Code smell detection and classification |
| `tech_debt.py` | Technical debt identification and tracking |
| `clone_detector.py` | Duplicate/cloned code detection |
| `naming_audit.py` | Naming convention audit |
| `comment_audit.py` | Comment quality and coverage audit |
| `dead_code_eliminator.py` | Dead code detection and removal |
| `import_optimizer.py` | Import organization and cleanup |
| `style_matcher.py` | Code style matching and enforcement |
| `type_adder.py` | Type annotation generation |
