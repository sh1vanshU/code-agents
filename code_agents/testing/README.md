# Testing

Testing tools, mutation testing, benchmarks, and test generation helpers.

## Modules

| Module | Description |
|--------|-------------|
| `mutation_testing.py` | Mutation testing framework |
| `mutation_tester.py` | Mutation test runner |
| `property_tests.py` | Property-based test generation |
| `test_style.py` | Test style and pattern analysis |
| `test_impact.py` | Test impact analysis for code changes |
| `test_fixer.py` | Automated test fix suggestions |
| `contract_testing.py` | API contract testing |
| `self_benchmark.py` | Agent self-benchmarking |
| `visual_regression.py` | Visual regression testing |
| `benchmark.py` | Performance benchmarking |
| `benchmark_regression.py` | Benchmark regression detection |
| `perf_proof.py` | Performance proof generation |
| `mock_builder.py` | Test mock/fixture generation |
| `edge_case_suggester.py` | Edge case test suggestions |
