# Git Ops

Git operations, PR management, and changelog tools.

## Modules

| Module | Description |
|--------|-------------|
| `git_hooks.py` | Git hook management and configuration |
| `git_story.py` | Git history storytelling |
| `pr_describe.py` | Automated PR description generation |
| `pr_split.py` | Large PR splitting suggestions |
| `pr_thread_agent.py` | PR comment thread agent |
| `diff_preview.py` | Diff preview and formatting |
| `blame_investigator.py` | Git blame investigation |
| `changelog.py` | Changelog generation |
| `release_notes.py` | Release notes generation |
| `action_log.py` | Action logging and audit trail |
